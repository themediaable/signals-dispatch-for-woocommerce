# WordPress.org Assets

Place plugin directory listing images here. They are synced to the
SVN `assets/` directory during release via `bin/wporg-release.sh`.

## Required Files

| Asset | Filename | Size |
|---|---|---|
| Banner (high-res) | `banner-1544x500.png` | 1544 × 500 |
| Banner (standard) | `banner-772x250.png` | 772 × 250 |
| Icon (high-res) | `icon-256x256.png` | 256 × 256 |
| Icon (standard) | `icon-128x128.png` | 128 × 128 |

## Optional

| Asset | Filename |
|---|---|
| Icon (SVG) | `icon.svg` |
| Screenshots | `screenshot-1.png`, `screenshot-2.png`, … |

Screenshot numbers must match the `== Screenshots ==` section in `readme.txt`.
